Análisis de Demanda Eléctrica ANDE

Descripción

Sistema de visualización y análisis de patrones de consumo eléctrico con comparativas históricas

Detalles

Plataforma completa para análisis energético con: - Carga automatizada de datos desde Excel - Procesamiento estadístico de series temporales - Visualizaciones interactivas con marcadores estacionales - Sistema de pruebas automatizadas - Gestión de dependencias con uv - Generación de reportes gráficos profesionales

Tecnologías Utilizadas

Python 3.13PandasMatplotlibSeabornOpenpyxlPytestuv

Carga de Datos desde Excel

            class DataLoader:
    @staticmethod
    def load_data(file_path):
        df = pd.read_excel(file_path)
        return df[['Año', 'Enero', 'Febrero', ..., 'Diciembre']]

Procesamiento de Datos

            def _process_data(self):
    self.df_melted = pd.melt(self.data, id_vars='Año',
        var_name='Mes', value_name='KWh')
    self.annual_stats = self.df_melted.groupby('Año')['KWh'].mean().to_dict()
